# 👤 Multi-Feature Face Processing System

A powerful Computer Vision application built with Python and OpenCV for face analysis and artistic image processing.

## 🚀 Deployment (Web)
This project is optimized for deployment on **Streamlit Cloud**, **Vercel**, or **Zeabur**.

**Entry point**: `web_app.py` (for Streamlit) or `main.py` (for general Python).

## 🛠️ Features
- **👤 Face Detection**: Identify and highlight faces.
- **🔒 Face Blur**: Anonymize individuals for privacy.
- **👥 Face Count**: Automatically count and label detected faces.
- **✨ Background Focus**: Professional depth-of-field effect.
- **🎨 Cartoonify**: Transform photos into artistic sketches.
- **🏁 Grayscale & 🎞️ Sepia**: Classic photographic filters.

## 💻 Local Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mohamednowfal-cyber/Multi-Feature-Face-Detection-cv.git
   cd Multi-Feature-Face-Detection-cv
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## 🖥️ How to Run
- **Desktop Version (Tkinter)**:
  ```bash
  python main.py
  ```
- **Web Version (Streamlit)**:
  ```bash
  streamlit run web_app.py
  ```

## 📜 Technical Stack
- **OpenCV**: Core Computer Vision engine.
- **Tkinter**: Native Desktop GUI.
- **Streamlit**: Modern Web Dashboard.
- **NumPy**: High-performance matrix operations.
- **Pillow**: Image format management.
